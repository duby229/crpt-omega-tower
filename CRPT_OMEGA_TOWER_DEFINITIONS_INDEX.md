## Part II — Notation Standardization Rules

All files in the ω-tower conform to these notation rules.

### Reduction Relation →_ρ

| Form | Usage | Notes |
|------|-------|-------|
| →_ρ | Base reduction (subscript always) | Never write →ρ |
| →_ρ* | Reflexive-transitive closure | Write as →_ρ* not →_ρ^* |
| →_ρ^n | n-step reduction | Superscript n |
| →_ρ^{Lift} | Reduction in Lift(M) | Superscript Lift (not subscript) |
| →_{ρ,M} | When model must be disambiguated | Used in L5+ when multiple models in scope |
| →_σ | Structural relation | Never mix with →_ρ |
| →_σ^{Lift} | Structural relation in Lift(M) | = →_ρ^{Lift} ∪ {commutativity} |

### Projection Operator ρ_M

| Form | Usage | Notes |
|------|-------|-------|
| ρ | L0 only — universal framework, no model | Never unsubscripted in L1+ |
| ρ_M | L1+ — projection of model M | Always subscript _M |
| ρ_{Lift(M)} | Projection of Lift(M) | Not ρ_Lift or ρ^{Lift} |
| ρ_{M_n} | Projection of n-th tower level | Not ρ_M^n (which means iteration) |
| ρ_M^n | n-fold composition of ρ_M | Superscript n = iteration count |

### Normal-Form Fiber NFC_M

| Form | Usage | Notes |
|------|-------|-------|
| NFC_M(f) | Always model-indexed | Never bare NFC(f) |
| NFC_M(f) | For convergent fibers | f ∈ Fix(ρ_M) ∩ ↓_M |

### Observable Equivalence ≃

| Form | Usage | Notes |
|------|-------|-------|
| ≃_π | L0 universal framework | w.r.t. projection π |
| ≃_M | L1+ CRPT models | Subscript _M always |
| ≃ | Never bare | Always subscripted |
| ≡ | Never used for observable equivalence | Reserved for definitional equality if needed |
| = | Mathematical identity | Distinct from ≃_M |

## Part IV — Axiom Profiles Quick Reference

The canonical axiom system has nine projection axioms (L1.2–L1.3). WF-side orbit
canonicalization (WF-Canon) is a theorem (L2), not an axiom.

| Profile | Required Axioms | Typical Use |
|---------|----------------|-------------|
| Pure-WF | PA-WN, PA-Conf, PA-Fix | Standard convergent models; Lift(M) is always Pure-WF |
| Native Stratified | PA-WN (scoped ↓_M), PA-NWF, PA-WN_top, PA-Reach | Both regimes ↓_M and ∞_M present |
| Pure NWF | PA-NWF, PA-CoInd, PA-WN_top | Persistent-only models (↓_M = ∅) |
| NWF-Prod | PA-NWF, PA-Prod | Productive persistent models |

**Key axiom meanings (the nine PA-* axioms):**
- **PA-WN:** Every element weakly normalizes (reaches Fix in finitely many ρ_M-steps) — applies to ↓_M
- **PA-Conf:** All terminating paths from any element converge to the same fixpoint
- **PA-Fix:** Fixed points are precisely normal forms: Fix(ρ_M) = NF(→_ρ)
- **PA-NWF:** Some elements have infinite orbits: ∞_M ≠ ∅
- **PA-CoInd:** Coinductive characterization of the persistent regime ∞_M
- **PA-Prod:** Each non-fixpoint projection step is observable (productivity)
- **PA-WN_top:** Persistent orbits converge topologically to a limit in 𝒯
- **PA-Bisim:** Bisimilar elements are orbit-equivalent (≈ ⊆ ≃_M)
- **PA-Reach:** On ∞_M, the orbit signature sig_M stabilizes under ρ_M-iteration (horizon stabilization)
